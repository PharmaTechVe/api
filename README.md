# PharmaTech Core API

## Project setup

```bash
$ npm install

# Prepare Husky
$ npm run prepare
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run the project with Docker
```bash
$ docker compose up --build
```

## Run tests

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:coverage
```
